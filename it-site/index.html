GRAFIČKI WEB DIZAJNER

Kurs za zanimanje "Grafički web dizajner"
podrazumeva savladavanje
materije iz oblasti web dizajna sa
ciljem kreiranja kompletnog dizajna i
organizacije web sajta, primenom
HTML i CSS-a. Ovaj kurs je proširen
naprednom upotrebom Photoshop-a
za pripremu, montažu i obradu slike sa
sajtove.

JAVA WEB DEVELOPER

Kurs za zanimanje Java Web Developer
osposobljava polaznika za razvoj web
aplikacija i Web servisa baziranih na
Java EE platformi. Na kursu se izučava
programski jezik Java, baze podataka i
jezik SQL, servleti, JSP, Spring,
Hibernate, web servisi, kao i
integracija sa web klijentskim
jezicima.

FRONT-END DEVELOPER

vo je najpopularniji kurs za
prekvalifikaciju za Front-End
Developera koji podrazumeva
kompletnu obuku za dizajniranje,
kreiranje i modifikaciju interaktivnih
web sajtova upotrebom HTML5 i CSS3
tehnologije gde je primarna upotreba
Java script-a, Ajax-a i jQuery-ja uz SEO
optimizaciju.

.NET DEVELOPER

Prekvalifikacija za .NET Developer-a
podrazumeva obuku za programiranje
u jednom od najmodernijih
programskih
jezika C# i razvoj
aplikacija za .NET platformu. Izučava
se programski jezik C# i .NET
biblioteka klasa, baze podataka i SQL, razvoj WPF desktop aplikacija, i web servisa.

BACK-END DEVELOPER

Prekvalifikacija za Back-End
Developer-a podrazumeva obuku
polaznika za najpopularni programski
jezik PHP kojim se kreiraju dinamičke
intenzivno se izučava i rad sa bazom
podataka MySQL, jezik SQL, kao i
integracija sa web klijentskim
jezicima.

ANDROID DEVELOPER

Prekvalifikacija za Android Developer-a
podrazumeva obuku korisnika za
kreiranje korisničkih aplikacija za
operativni sistem Android
korišćenjem programski jezik Java,
Android SDK objektno-orijentisane
biblioteke kao i XML jezika za
definisanje korisničkog interfejsa

MANUELNO TESTIRANJE

Testeri imaju jednu od ključnih uloga
u procesu razvijanja softvera. Njihov
zadatak je da spreče da greške
softveru dođu do krajnjih korisnika.
Prekvalifikacija za Testera
podrazumeva obuku polaznika za
kreiranje i implementaciju test – case-
ova upravo koji za cilj upravo imaju
minimiziranje grešaka.

LINUX ADMINISTRATOR

Prekvalifikacija za Linux
administratora podrazumeva efikasno
korišćenje i administiranje računara
koji koriste jedan od najpopularnijih
operativnih sistema – Linux, na
primerima različitih distribucija.
Poseban akcenat je na optimizaciji
performansi i bezbednosti servera u
Linux okruženju.
